# grafana-annotation

在 Grafana 中创建注解的插件。

## 在 云原生构建 上使用

```yml
master:
  push:
  - stages:
    - name: annotate
      image: fdeschenes/drone-grafana-annotation
      settings:
        api_key: xxxxx
        image: fdeschenes/drone-grafana-annotation
        tags:
          - deploy
          - production
        text: "Deployed"
        url: https://grafana.example.com

```

## 参数含义

api_key
: api key。

tags
: tag 列表。

text
: 描述。

url
: 服务器 url。
