# Jenkins

触发 Jenkins Jobs 的插件。

## 在 Coding-CI 上使用

```yml
main:
  push:
  - stages:
    - name: trigger jenkins job
      image: appleboy/drone-jenkins
      settings:
        url: http://example.com
        user: appleboy
        token: xxxxxxxxxx
        job: drone-jenkins-plugin-job
```

## 参数含义

url
: jenkins 服务地址。

user
: jenkins 用户账号。

token
: jenkins 用户 token。

job
: jenkins job 名称。
